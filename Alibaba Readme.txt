Alibaba Stock Data Analysis


About
This project aims to explore the historical stock price data for Alibaba Group (NYSE: BABA)

Purposes Of The Project
The major aim of this project is to gain insight into the historical stock price for Alibaba Group to understand the different factors that affect stock performance

About Data
The dataset was obtained from the https://www.kaggle.com/datasets/mhassansaboor/alibaba-stock-dataset-2025/data

Trend Analysis: Analysis long-term trends in Alibaba's stock prices to identify growth patterns or periods of decline.
Volatility Assessment: Analysis of changes in daily prices to assess the stock's volatility over different time frames.
